## 页面功能说明

1. **主页面**：
   - 顶部显示页面标题"用户模型管理"
   - 主要内容是一个详细的用户模型列表表格
   - 表格包含以下列：模型名称、模型编码、提供商、模型标识、API URL、API Key、描述、状态、创建时间、更新时间
   - API Key 列显示"查看"按钮，点击可打开对话框查看完整 API Key
   - 状态列使用不同颜色的标签显示启用/禁用状态
   - 时间列显示格式化的日期时间

2. **API Key 查看对话框**：
   - 显示标题"API Key"
   - 包含一个只读输入框，显示完整的 API Key
   - 底部有一个"关闭"按钮

3. **操作功能**：
   - 查看详细的 API Key
   - 格式化显示日期时间
