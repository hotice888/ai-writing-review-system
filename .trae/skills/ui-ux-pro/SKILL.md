# UI/UX Pro

## 功能

该技能提供设计智能，用于构建专业的 UI/UX 跨多个平台和框架，包括：

- 智能设计系统生成
- 专业界面设计
- 色彩方案推荐
- 排版系统建议
- 响应式布局设计
- 交互效果优化

## 使用方法

当用户需要以下功能时，可以调用此技能：

1. 生成完整的设计系统
2. 设计专业的用户界面
3. 获取色彩方案和排版建议
4. 优化交互效果和用户体验
5. 构建响应式布局

## 设计模式

### 1. Hero-Centric
- **核心**：大型英雄区域，突出品牌信息和主要CTA
- **适用场景**：品牌网站、产品着陆页、服务介绍
- **结构**：Hero区域 → 特性/服务 → 社会证明 → CTA

### 2. Content-First
- **核心**：内容驱动，清晰的信息层次
- **适用场景**：博客、新闻网站、教育平台
- **结构**：导航 → 内容列表 → 详细内容 → 相关内容

### 3. Dashboard
- **核心**：数据可视化和快速操作
- **适用场景**：管理系统、分析工具、控制面板
- **结构**：侧边导航 → 顶部栏 → 数据卡片 → 详细图表

### 4. E-Commerce
- **核心**：产品展示和便捷购买流程
- **适用场景**：在线商店、购物平台
- **结构**：首页英雄区 → 产品分类 → 产品列表 → 购物车 → 结账

## 设计风格

### 1. Soft UI Evolution
- **关键词**：柔和阴影、微妙深度、平静、高级感、有机形状
- **适用场景**：健康、美容、生活方式品牌、高级服务
- **性能**：优秀 | 可访问性：WCAG AA

### 2. Modern Minimalism
- **关键词**：干净线条、充足留白、功能性、清晰层次
- **适用场景**：科技产品、 SaaS、专业服务
- **性能**：优秀 | 可访问性：WCAG AA

### 3. Bold & Dynamic
- **关键词**：大胆色彩、强烈对比、动态元素、现代感
- **适用场景**：创意机构、数字产品、年轻人品牌
- **性能**：良好 | 可访问性：WCAG AA

### 4. Neo-Brutalism
- **关键词**：原始美学、大胆形状、高对比度、非常规布局
- **适用场景**：创意品牌、艺术项目、实验性网站
- **性能**：优秀 | 可访问性：需注意

## 交互效果

### 1. 微妙动画
- **过渡时间**：200-300ms
- **效果**：柔和的悬停状态、平滑的页面过渡
- **适用场景**：大多数现代网站和应用

### 2. 响应式反馈
- **点击反馈**：轻微的缩放或颜色变化
- **滚动效果**：视差滚动、渐入渐出
- **适用场景**：互动性强的网站、产品展示

### 3. 微交互
- **图标动画**：悬停时的细微变化
- **表单反馈**：实时验证和提示
- **适用场景**：复杂应用、用户仪表盘

## 预交付检查清单

- [ ] 无表情符号作为图标（使用SVG：Heroicons/Lucide）
- [ ] 所有可点击元素都有 cursor-pointer
- [ ] 悬停状态带有平滑过渡（150-300ms）
- [ ] 浅色模式：文本对比度最低4.5:1
- [ ] 键盘导航的焦点状态可见
- [ ] 尊重 prefers-reduced-motion
- [ ] 响应式：375px、768px、1024px、1440px
- [ ] 性能优化：图片压缩、代码最小化
- [ ] 跨浏览器兼容性测试
- [ ] 可访问性测试（WCAG AA标准）
