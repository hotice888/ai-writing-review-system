# ASCII Design

## 功能

该技能用于快速设计界面原型，利用纯文本字符绘制低保真页面草图，通过视觉化排版传达页面结构。它可以帮助用户：

- 快速构思和展示页面布局
- 传达界面结构和组件关系
- 在开发前进行设计沟通
- 生成简洁明了的视觉参考

## 使用方法

当用户需要以下功能时，可以调用此技能：

1. 设计新页面的布局原型
2. 绘制现有页面的结构草图
3. 展示界面组件的排列方式
4. 快速传达设计思路

## 绘制规则

1. 使用 `+`, `-`, `|` 等字符构建边框和分隔线
2. 使用 `[ ]` 表示按钮和可点击元素
3. 使用 `___________` 表示输入框
4. 使用缩进和对齐来表示层次结构
5. 保持代码块的格式清晰易读

## 输出格式

输出应包含：

1. 页面名称或功能描述
2. ASCII 绘制的页面草图
3. 简要的结构说明（可选）

## 应用场景

- 快速设计原型
- 团队内部设计沟通
- 开发前的布局规划
- 技术文档中的界面说明
