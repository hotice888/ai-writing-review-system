
# 趋势统计功能修改记录

## 功能概述
- **文件位置**: `adminFrontend/src/views/StatisticsDashboard.vue`
- **功能描述**: 统计仪表盘的Token使用趋势图表，支持按天/周/月/年统计，支持总Token、业务类型、模型标识、模型平台四个标签页

## 核心功能
1. **周期统计**: 支持天、周、月、年四种周期类型
2. **多标签页**: 总Token、业务类型、模型标识、模型平台
3. **图表展示**: 柱状图（总Token）、折线图（多类别）
4. **全屏查看**: 支持双击图表放大查看
5. **数据过滤**: 支持日期范围选择、周期类型切换
6. **Tooltip提示**: 鼠标悬停显示详细数据

## 技术实现
- **框架**: Vue 3 + TypeScript
- **UI组件**: Element Plus
- **图表**: SVG原生绘制（非第三方图表库）

## 关键常量配置
```javascript
const chartHeight = 200;              // 主图表高度
const dialogChartHeight = 300;          // 对话框图表高度
const periodWidth = 40;               // 周期宽度
const visiblePeriodsCount = 12;        // 可见周期数
```

## 数据计算逻辑
1. **Y轴刻度: 7个刻度（0, 0.2max, 0.4max, 0.6max, 0.8max, max, 1.2max）
2. **数据点**: y坐标 = chartHeight + 20 - (value / (max*1.2) * chartHeight
3. **横轴总宽度**: Math.max(实际周期数, visiblePeriodsCount) * periodWidth

## 注意事项
1. SVG的viewBox需要足够大以避免内容被裁剪
2. 主图表和对话框图表保持一致的逻辑
3. 滚动同步确保主图表和对话框图表
4. 周期标签换行使用tspan而非\n
5. 坐标轴交叉在(0, chartHeight+20)
